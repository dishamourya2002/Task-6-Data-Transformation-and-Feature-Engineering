# Task 6: Data Transformation & Feature Engineering

## Objective

Perform feature scaling and feature engineering on the California Housing Dataset.

## Tools Used

* Python
* Pandas
* Scikit-learn
* Google Colab

## Steps Performed

1. Loaded the California Housing Dataset.
2. Explored the dataset using `info()` and `describe()`.
3. Applied StandardScaler for feature scaling.
4. Created a new feature: `Income_Rooms = MedInc × AveRooms`.
5. Saved the transformed dataset as CSV.

* Task6_Feature_Engineering.ipynb
* engineered_housing_dataset.csv
* Task6_Report.pdf

## Outcome

Successfully transformed the dataset and created a new feature to improve machine learning readiness.
